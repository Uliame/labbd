# Projeto LabBD - Visualizador de Vagas e Currículos

Um aplicativo web simples construído com Streamlit para visualizar e listar dados de currículos e vagas de emprego a partir de arquivos CSV. Este projeto foi desenvolvido como parte de um estudo ou laboratório de banco de dados (LabBD).

## Funcionalidades

    Visualizador de Currículos: Carrega e exibe o arquivo curriculos.csv em uma tabela interativa.

    Visualizador de Vagas: Carrega e exibe o arquivo vagas.csv em uma tabela interativa.

## Tecnologias Utilizadas

    Python: Linguagem de programação principal.

    Streamlit: Framework para a criação da interface web interativa.

    Pandas: Biblioteca para leitura e manipulação dos dados dos arquivos CSV.
